Instalação

Guia rápido para os jogadores

  1. Baixe o arquivo principal e extraia para uma pasta sem espaços, por exemplo C:\\ConquerRedux.
  2. Se houver patch, extraia por cima do cliente substituindo os arquivos.
  3. Execute o launcher ou play.exe e mantenha o antivírus como exceção se necessário.

Checklist antes de publicar

  • Envie o cliente para /WebTemplate5065/downloads/ ou um CDN configurado
  • Atualize os hrefs desta página com as URLs finais
  • Teste o download direto e via espelho em um navegador limpo
  • Divulgue o hash do arquivo para evitar downloads corrompidos
